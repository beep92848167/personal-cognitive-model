# Scoring

Each recommendation should include:

- Taste Fit: how well it matches durable preferences.
- Cognitive Fit: how suitable it is for the user's current energy/cognitive load.
- Confidence: how sure the AI is.
- Cautions: why it might fail.

Suggested weighting:

- writing/story quality: 30%
- systems/institutions/stewardship: 25%
- consistency/pacing: 15%
- dialogue/characters: 15%
- genre/creator bonus: 15%
