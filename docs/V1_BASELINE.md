# OpenPCM V1 Baseline

Status: Frozen
Baseline commit: `f5f82a2`

OpenPCM V1 is considered complete at the `f5f82a2` baseline.

## V1 completion criteria

- 24/24 requirements covered.
- 57/57 automated tests passing.
- Android / Termux workflow documented and tested.
- Browser and CLI test runners aligned.
- Personal Cognitive Model seed integrated into Discover.
- Discover provides explainable recommendations.
- Recommendation calibration is implemented.
- Repository handoff and SYNC workflow are documented.

## V1 policy

V1 is now bug-fix only.

Acceptable V1 changes:

- critical bug fixes;
- data-loss prevention;
- security/privacy fixes;
- documentation corrections;
- workflow breakage fixes.

Non-V1 changes should be treated as V2 work.

## Why freeze V1?

Freezing V1 creates a stable, fully tested baseline before expanding OpenPCM into a broader personal reasoning system.
